Thatsa Mortgage

neumorphism | angular | SASS | UI/UX

Thatsa Mortgage (stylized as thatsa Mortgage) is a simple mortgage calculator built to practice my skills as well as incorporate new design ideas. It's built using the Angular framework, SASS, and Neumorphism UI styling.

This project let me fully explore my ideas from start to finish in a way I hadn't done before - totally alone. Being able to breakdown every component of the development process, from wireframe to final build, helped me to understand steps and set backs of a project in a new light.
