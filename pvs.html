Peeple vs Sheeple

angular | heroku | web development

Peeple Vs. Sheeple is a 'would-you-rather' style game where players choose one of two options to see if they answer in the miniority (people) or the majority (sheeple). It's a simple game that my team and I built with UI in mind, but with some extra bells and whistles like a high-score page and the ability to add your own questions.

PvS was built using Angular framework and with Heroku database management to hold all the questions and scores. PvS also uses a few web API to populate a random avatar and quote to add levity (it as a game after all).
